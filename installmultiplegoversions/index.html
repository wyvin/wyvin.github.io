安装多个Go版本

系统：MacOS 原有的版本为：1.13.6 增加额外版本：1.16.3

设置代理

直接下载会遇到网络问题，需要设置一下代理

1
2

⇒  export GO111MODULE=on
⇒  export GOPROXY=https://goproxy.cn

使用

再做个版本差异的验证

在1.15版本的特性中：将较小的整数值转换为接口值的过程中不再会引起内存分配

import (
	"testing"
)

func Convert(val int) []interface{} {
	var slice = make([]interface{}, 100)
	for i := 0; i < 100; i++ {
		slice[i] = val
	}

	return slice
}

func BenchmarkConvert(b *testing.B) {
	for i := 0; i < b.N; i++ {
		result := Convert(254)
		_ = result
	}
}

卸载

  • 删除$GOPATH/bin/go1.16.3
  • 删除/sdk/go1.16.3
